Set a value of a data message field.

This base implementation simply logs a warning, and ignores the field. Derived classes should override this implementation to set the relevant class members.

Parameters

dataThe raw data to extract the field value from.
baseTypeThe FIT base type for the field.

Returns
true if the field was set, or skipped; false if the field was corrupt or invalid.
